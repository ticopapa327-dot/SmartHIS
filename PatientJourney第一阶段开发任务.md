# Patient Journey 第一阶段开发任务

版本：0.1  
日期：2026-05-16

## 1. 目标

实现一个可运行的胆囊结石择期住院手术旅程，让 SmartHIS 控制台可以展示并推进病人从入院到出院的全过程。

## 4. 当前 MVP 说明

当前第一阶段以 `PAT000001 / ENC000001 / SCH000001` 为默认演示病人，模板为 `TPL_CHOLECYSTECTOMY_INPATIENT`。

旅程可通过以下 API 操作：

```http
GET /api/v1/patient-journeys
POST /api/v1/patient-journeys/JNY000001/next
POST /api/v1/patient-journeys/JNY000001/run
GET /api/v1/patient-journeys/JNY000001/timeline
```

远端部署后可在控制台操作：

```text
http://192.168.1.60:7070/console
```
